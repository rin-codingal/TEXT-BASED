# Sample Inputs – AI Prompt Engineering Tutorial

Use the following **sample inputs** to test the program flow where prompts move from **vague → specific → contextual**.  
When running the code, copy and paste **one set at a time** as the program asks for inputs.

---

## Sample Set 1: General Knowledge

### Vague Prompt
```
Tell me about planets.
```

### Specific Prompt
```
Explain the difference between inner and outer planets in our solar system.
```

### Contextual Prompt
```
Explain the difference between inner and outer planets in simple words for a 10-year-old, using 3 bullet points and 1 fun example.
```

---

## Sample Set 2: Coding (Scratch Game Idea)

### Vague Prompt
```
Write code for a game.
```

### Specific Prompt
```
Write a Scratch game idea where the player collects coins and avoids obstacles.
```

### Contextual Prompt
```
Write a Scratch game idea for Grades 6–8 where the player collects coins, loses 1 life on hitting obstacles, and the game ends at 0 lives. Include scoring and one power-up idea. Explain in 6 short steps.
```

---

## Sample Set 3: Real-World Writing (Parent Email)

### Vague Prompt
```
Write an email.
```

### Specific Prompt
```
Write an email to a parent about a student's good progress in coding.
```

### Contextual Prompt
```
Write a short, polite email to a parent of a Grade 5 student. Mention that the student completed a Scratch animation project, improved logical thinking, and should practice coding for 15 minutes daily. Keep it under 120 words.
```

---

## Learning Tip
> Each step improves the prompt by adding **clarity**, **specific details**, and **context**, leading to more accurate and useful AI responses.
